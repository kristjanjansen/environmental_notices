### Installation

```
npm install
```

Then create a config/config.json file:

```
{
  "baseUrl": "site_base_url",
  "mailFrom": "mail_to_send_from",
  "mailTo": [
    "mail_to_send_to_1",
    "mail_to_send_to_2"
  ],
  "mailUsername": "gmail_account_to_sent_from",
  "mailPassword": "gmail_account_password"
}
```
